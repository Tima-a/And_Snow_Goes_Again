# And Snow Goes Again
And snow goes again... is a 3D-horror adventure, puzzle game published on itch.io.
# Game mechanics, rules
And snow goes again is a first person view arcade and horror game. After 10 minutes player will die, to escape death player will spin, wander around the forest and gather coins and batteries for the flashlight. User can convert every coin into 30000$ through the ATM machine. From the start of the game you have 80000$. To escape should spin the wheel of fortune which requires 60000$ and 10000$ additionally every time you spin the wheel again. When player spins the wheel of fortune current timeline changes and rewinds the time to 10 minutes earlier. Player has to find the door in dark dimension and change time current timeline. Player's goal is to get 8 coins and spin the wheel simultaneously. But there are two rules, player must have more than 65001$ every round and this value increases by 5000$ every time you alter the current timeline. Furthermore, if player finds the teddybear after the third round you will get additional 50000$. You should not forget about the time and gathering the coins in the forest, sometimes you must think rapidly and critically to pass the game.
